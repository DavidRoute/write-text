## Quick Start
- Clone or download this repo and place it into your server.
- `composer install `
- `cp .env.example .env `
- Create database and modify .env with your DB name.
- `php artisan key:generate `
- `php artisan migrate `
- `php artisan db:seed `
- `php artisan sync:holidays `
- `php artisan storage:link`
- `php artisan serve `
